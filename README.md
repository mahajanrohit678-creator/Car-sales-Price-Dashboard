# 🚗 Car Sales Price Dashboard

## Overview
Interactive Excel dashboard for analyzing car sales data. Built using Pivot Tables, Pivot Charts, KPIs, and Slicers to provide insights into pricing, mileage, fuel types, brands, and transmission distribution.

## Features
- Total Cars KPI
- Average Price Analysis
- Maximum Price Tracking
- Average Mileage Analysis
- Fuel Type Comparison
- Model Year Trends
- Brand Distribution
- Transmission Distribution
- Interactive Filters

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Visualization
